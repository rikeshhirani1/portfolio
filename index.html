User Behaviour Analytics Tool


The aim of this university project was to develop a library that can track user’s interactions within a website. The data was available to be viewed and interpreted using visualisations. Data recorded included the user’s page navigation route, mouse cursor movement and mouse clicks (pointer placement). The data gathered and visualised aided developers to improve user experience and optimise their website dependent on how users interact with their website.
Responsive data visualisations were created using HTML5 Canvas for each recorded metric.

Technologies and Services Used:
Node.js, Express, JavaScript, jQuery, HTML, Bootstrap, MongoDB, GitHub and Mocha.
